# 短期上下文使用说明

在 Conversation 窗口顶部点击“上下文”图标可打开检查器。

## 检查器

检查器显示预计总 Token、输入占用、输出预留、上下文上限，以及基础提示词、Personality、长期记忆、摘要、固定消息和最近对话的分项占用。

## 消息操作

悬停一条消息可执行：

- 不加入上下文 / 重新加入上下文
- 固定到本会话 / 取消固定
- 复制

排除消息时会同步取消固定。

## 会话摘要

摘要由用户手动维护，适合记录当前目标、已完成事项、关键决定和下一步。摘要只在当前会话生效。

## 清除当前上下文

此操作保留所有历史消息，只让下一次模型请求从清除点后重新开始。固定消息和手动摘要不会被清除。
